🎨 UI Utilities🎨 Utilidades de UI

Seven helpers for the day-to-day of building JavaFX screens: theming, dialogs, tables, input formatting, two-way binding and image handling. Each is self-contained and adds no external dependency. Siete ayudantes para el día a día de construir pantallas JavaFX: theming, diálogos, tablas, formateo de entrada, binding bidireccional y manejo de imágenes. Cada uno es autónomo y no agrega ninguna dependencia externa.

ThemeManager
Register named CSS theme sets, activate one, and apply it to scenes. Live-switches all managed scenes; tracks them with weak references.Registra conjuntos de temas CSS con nombre, activa uno y aplícalo a las escenas. Cambia en vivo todas las escenas gestionadas; las rastrea con referencias débiles.

AlertUtil
Themed alert/confirmation dialogs with self-contained CSS injected via a data: URI. Presets, a color builder, or raw CSS. Recommended for new code.Diálogos de alerta/confirmación con tema y CSS autónomo inyectado mediante un data: URI. Presets, un builder de colores o CSS en crudo. Recomendado para código nuevo.

Message

AlertUtil vs. MessageAlertUtil vs. Message

Both show the same dialogs. Reach for AlertUtil when you want theming (dark UIs, branded colors); reach for Message when you want the lightest, theme-free option. They share identical content sizing.Ambos muestran los mismos diálogos. Usa AlertUtil cuando quieras theming (interfaces oscuras, colores de marca); usa Message cuando quieras la opción más ligera y sin tema. Comparten el mismo dimensionado de contenido.
